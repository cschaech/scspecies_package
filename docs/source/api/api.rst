API Reference
=============

scspecies contains three submodules. 

- `scspecies.preprocessing` contains functions and classes to download and set up context and target datasets such that they can be used by **scSpecies**.
- `scspecies.models` contains functions and classes used by the the main **scSpecies** model.
- `scspecies.plot` contains functions to visualize the results of a fitted model.
